### Let's try and make some more interesting and custom animations

#### A `motion` can have multiple props defining some things, let's have a look...

A motion to have an element fade in...

<motion.div
  initial={{
    x: -100 * toRight,
    opacity: 0,
  }}
  whileInView={{ x: 0, opacity: 1 }}
  viewport={{ once: true }}
  transition={{
    x: { type: "spring", duration: 0.8, delay: 0.2 },
    opacity: { delay: 0.2 },
  }}
>
```jsx
<motion.div
  initial={{ opacity: 0 }}
  animate={{ opacity: 1 }}
/>
```
</motion.div>

A motion to move an element to it's correct position...

<motion.div
  initial={{
    x: -100 * toLeft,
    opacity: 0,
  }}
  whileInView={{ x: 0, opacity: 1 }}
  viewport={{ once: true }}
  transition={{
    x: { type: "spring", duration: 0.8, delay: 0.2 },
    opacity: { delay: 0.2 },
  }}
>
```jsx
<motion.div
  initial={{ x: -100, y: 100 }}
  animate={{ x: 0, y: 0 }}
/>
```
</motion.div>

Not only can you move and change the opacity, you can add some transition effects to how your element should move to it's position...

<motion.div
  initial={{
    x: -100 * toRight,
    opacity: 0,
  }}
  whileInView={{ x: 0, opacity: 1 }}
  viewport={{ once: true }}
  transition={{
    x: { type: "spring", duration: 0.8, delay: 0.2 },
    opacity: { delay: 0.2 },
  }}
>
```jsx
<motion.div
  initial={{ opacity: 0, x: -100, y: 100 }}
  animate={{ opacity: 1 x: 0, y: 0 }}
  transition={{
    x: { type: "spring", duration: 0.8, delay: 0.2 },
    opacity: { delay: 0.2 },
  }}
/>
```
</motion.div>

Motion is now your oyster, be creative and start playing with animations.
